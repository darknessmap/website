The Darkness Map is a visualization of light levels in the city at night. To collect this data, we made a custom mobile app that computes the average brightness for each frame of a video clip, and graphs that value according to the frame's GPS location.

Add to the Map.

Once the Darkness Mapp application is downloaded to your mobile device (beta version coming soon for the Android OS), activate it at night as you walk around your neighborhood. Video is GPS tagged and processed to output data that is uploaded to the Darkness Map servers and added to the map.

How it works.

The Darkness Map application finds the average brightness of each frame of video then graphs that value as a vertical line to create a histogram. Each frame is tagged with a GPS location so it can easily be added to the map.
